Three-layer security architecture ensuring end-to-end protection from user input to server transmission. Each keystroke is independently encrypted with random padding to prevent pattern analysis.

Layer 1: Input Protection
Each keystroke encrypted with AES-256/CBC. Random 15-byte padding per input prevents frequency analysis even when same key pressed multiple times.
AES-256-CBC ‚Ä¢ PKCS5Padding ‚Ä¢ 16-byte blocks
Layer 2: Key Exchange
Symmetric session key encrypted with RSA-2048 public key. Key loaded from assets (vkeypad_public.pem) ensures secure key distribution.
RSA-2048 ‚Ä¢ PKCS1Padding ‚Ä¢ 256-byte asymmetric encryption
Layer 3: Integrity Verification
HMAC-SHA1 signature validates data integrity during transmission. Detects tampering and ensures authentic encrypted payload delivery.
HMAC-SHA1 ‚Ä¢ 20-byte MAC ‚Ä¢ Tamper detection
